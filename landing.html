Everything you need to understand your users

Seven core capabilities, one lightweight script. Built for teams that ship fast and need answers now.

Embed Script

Lightweight JS snippet under 1KB gzipped. One-line install, no cookies by default, fully GDPR-compliant. Auto-captures page views and clicks out of the box. The entry point for all data collection including error tracking.

Event Tracking

Auto-capture page views, clicks, and form submissions. Custom event API for tracking business-specific actions like signups and purchases. Retroactive event definition from captured data — you never miss a signal.

Dashboard

Real-time analytics dashboard showing visitors, unique users, sessions, bounce rate, top pages, referrers, UTM breakdown, and geographic data. Customizable date ranges, filters, and comparison periods.

Funnels

Define multi-step conversion funnels and visualize drop-off at each step with exact percentages. Filter by user properties, time range, or cohort. Compare funnel performance across segments to find what converts.

Error Tracking

Sentry-like error capture: automatic JS error collection via embed script, stack traces with source map support, error grouping and deduplication, severity levels, and alert notifications via webhook or email.

User Sessions

Session replay with lightweight DOM snapshots (not video recording). User journey visualization showing page flow and navigation paths. Click heatmaps on any page to see exactly where attention goes.

API Access

REST API for querying all analytics data programmatically. Webhook support for real-time event forwarding. API key authentication with rate limiting. Integrate your analytics data into any workflow or tool.
