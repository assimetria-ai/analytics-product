Features

Everything you need to understand your users

From a lightweight embed script to full session replay and error tracking, built for teams that care about privacy and performance.

Embed Script

Lightweight JS snippet under 1KB gzipped. One-line install, no cookies by default, fully GDPR-compliant. Auto-captures page views and clicks out of the box.

Event Tracking

Auto-capture page views, clicks, and form submissions. Custom event API for business actions. Retroactive event definition from captured data — never miss an event.

Real-Time Dashboard

Visitors, sessions, bounce rate, top pages, referrers, UTM breakdown, and geo data — all in real time. Customizable date ranges, filters, and comparison periods.

Conversion Funnels

Define multi-step funnels and visualize drop-off at each step. Filter by user properties, time range, or cohort. Compare performance across segments.

Error Tracking

Sentry-like error capture built in. Automatic JS error collection, stack traces with source map support, grouping, deduplication, severity levels, and alert notifications.

User Sessions

Session replay with lightweight DOM snapshots. User journey visualization with page flow and navigation paths. Click heatmaps on any page.

API Access

REST API for querying all analytics data programmatically. Webhook support for real-time event forwarding. API key authentication with rate limiting.
